# Face Detector Examination System

A Python-based examination monitoring tool that uses computer vision to detect face-related exam violations and save local evidence screenshots.

## Project Files

- `exam detection/TheFaceDetector.py` - main face detection and monitoring script
- `exam detection/exam_evidence/` - generated evidence screenshots, ignored by Git

## Notes

Evidence photos and other generated media are intentionally excluded from the repository for privacy and cleanliness.
